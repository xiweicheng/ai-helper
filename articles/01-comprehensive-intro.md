## 引言：浏览器，AI 时代的终极工作台

每天，我们在浏览器里阅读文档、处理数据、写代码、填表单、查资料。而现有的 AI 工具要么是独立的网页应用，要么只能被动地回答问题——它们无法真正"理解"你正在看的网页内容，更无法替你操作浏览器。

**AI Helper 改变了这一切。**

它是一个开源的 Chrome 浏览器智能助手扩展，将大语言模型的推理能力与浏览器的原生能力深度结合，让浏览器真正学会"思考"和"行动"。

---

## 一句话概括

> AI Helper = ChatGPT 的对话能力 + 浏览器的操作能力 + 自定义智能体的协作能力 + 本地文件/命令的执行能力，全部装在侧边栏里。

---

## 核心亮点速览

### 1. ReAct 推理循环——让 AI 学会"思考-行动"循环

传统 AI 对话是"你问一句，我答一句"。AI Helper 采用了 **ReAct（Reasoning + Acting）** 架构：

```
用户任务 → AI 思考 → 决定调用工具 → 执行工具 → 分析结果 → 继续推理 → 完成任务
```

这意味着你只需说"帮我把这个网页的表格数据导出为 Excel"，AI 会自动完成：识别表格 → 提取数据 → 格式化 → 触发下载，全程无需人工干预。

**关键技术细节：**
- 工具预筛选：从 60+ 工具中智能选择 5-10 个相关工具，大幅节省 Token
- 动态 Token 预算管理：按模型上下文窗口实时调整
- 并行工具执行：同一轮的独立工具调用并发执行
- 全流程可视化：执行日志面板实时追踪每一步推理

### 2. 62 个内置工具——浏览器即操作系统

AI Helper 内置了 9 大类共 62 个工具，覆盖浏览器能力的方方面面：

| 类别 | 工具数量 | 代表能力 |
|------|----------|----------|
| 内容提取 | 16 | 页面转 Markdown、表格识别、文本提取 |
| 页面交互 | 7 | 点击元素、表单填写、拖拽操作 |
| 标签页管理 | 6 | 打开/切换/关闭标签页 |
| 网络请求 | 1 | HTTP 请求发送 |
| 媒体与输出 | 7 | 截图、二维码生成、文件下载 |
| AI 协作 | 5 | 子任务分发、任务规划、UI 原型预览 |
| 存储管理 | 3 | Cookie 管理、数据清理 |
| 书签与历史 | 2 | 书签搜索、历史搜索 |
| 调试与开发 | 3 | CSS 注入、浏览器信息获取 |

### 3. 多智能体协作——一个插件，多个 AI 专家

内置 **5 个专业助手模板**，覆盖不同场景：

- **默认助手** — 通用对话与网页问答
- **代码审查专家** — 代码质量、安全、最佳实践审查
- **网页自动化助手** — 表单填写、数据抓取、页面操作
- **数据分析师** — 网页数据提取、统计分析、可视化
- **文档撰写助手** — 技术文档、报告、总结撰写

支持用户创建自定义 Agent，为每个 Agent 配置独立的系统提示词和工具权限。通过 `@` 符号快速切换，一个任务可以自动分发给最合适的 Agent 处理。

### 4. 可选代理服务——突破浏览器沙箱

浏览器的安全沙箱限制了对本地文件系统和命令行的访问。AI Helper 提供了可选的 **Node.js 本地代理服务**，解锁以下能力：

- **文件系统操作**：读取/写入/搜索/删除本地文件
- **命令执行**：运行终端命令，支持实时输出流（WebSocket）
- **Skill 系统**：将常用工作流沉淀为可复用的技能
- **MCP 协议扩展**：连接第三方工具服务，无限扩展能力边界

安全方面采用路径沙箱 + 黑白灰名单 + 操作确认的多层防护。

### 5. 划词即用——不打断工作流

在任意网页选中文本，浮动工具栏立即出现：

- AI 搜索、解释、翻译、总结一键触发
- 自定义提示词工具栏，拖拽排序
- 内联追问，无需切换到侧边栏
- 结果面板可拖拽、可缩放

### 6. 质量保障——三级反思系统

AI 不是完美的，AI Helper 内置了三级质量检查：

1. **工具级反思**：每次工具调用后评估结果有效性
2. **子任务反思**：复杂子任务的完整性和相关性评估
3. **后置反思**：最终答案的 **7 维度评分**（完整性/准确性/相关性/工具使用/清晰度/安全性/效率），低于阈值自动修订或重试

### 7. 更多亮点

- **Markdown + Mermaid 图表渲染**：AI 输出的流程图、时序图直接渲染
- **UI 原型预览**：AI 生成的 HTML/CSS 内联预览并保存到原型库
- **Token 统计面板**：实时追踪消耗，趋势图展示，上下文压力监测
- **多会话管理**：标签式切换，配置独立保存
- **30+ 大模型支持**：兼容 OpenAI API 格式，DeepSeek/通义千问/GPT-4 等即配即用

## 立即开始

1. 克隆项目，`npm install && npm run build`
2. Chrome 打开 `chrome://extensions/`，加载 `dist` 目录
3. 配置 API Key，选择模型，开始使用
