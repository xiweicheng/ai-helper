## 推广平台推荐优先级

### 第一梯队（必发）
- **掘金** — 国内最大开发者社区，技术文章流量大
- **知乎** — 自带搜索权重，长尾流量持续

### 第二梯队（强烈推荐）
- **V2EX** — 高质量技术讨论，种子用户聚集
- **少数派** — 效率工具垂直社区，受众精准
- **SegmentFault** — 技术问答+文章，SEO 友好

### 第三梯队（补充覆盖）
- **CSDN** — 流量大但竞争激烈
- **博客园** — 老牌技术社区
- **即刻** — 年轻用户社区
- **小红书** — 效率工具话题有红利

### 海外推广
- **Product Hunt** — 产品首发必选
- **Twitter/X** — 开发者社群活跃
- **Reddit** — r/chrome、r/chatgpt、r/selfhosted
- **Hacker News** — 技术产品曝光
- **Dev.to** — 英文技术文章

## 发布建议

1. **首发时机**：选择周二-周四上午 10:00 发布，流量最佳
2. **交叉引流**：每篇文章末尾附上其他平台链接，形成流量闭环
3. **互动维护**：发布后 2 小时内积极回复评论，提升算法推荐
4. **SEO 优化**：标题包含核心关键词（AI 浏览器助手、Chrome 扩展、开源）
5. **配图规范**：首图使用 1200x630 尺寸，正文配 GIF 动图展示功能
