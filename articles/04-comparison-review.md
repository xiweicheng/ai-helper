| Monica | ✅ 选中文本 | ❌ | ✅ | ❌ |

**结论**：AI Helper 是唯一一个能对页面进行"全量理解"的工具——不仅读取文本，还能识别表格结构、提取结构化数据、将页面转为 Markdown。

### 1.2 浏览器操作能力（关键差异）

| 产品 | 页面交互 | 表单填写 | 标签管理 | 截图/下载 |
|------|:---:|:---:|:---:|:---:|

## 四、适用场景推荐

### 选 AI Helper 如果你：
- 需要让 AI 实际操作浏览器（填表、点击、切换标签页）
- 需要网页数据提取和自动化处理
- 需要执行本地文件操作或命令
- 关注数据隐私，不想经过第三方服务器
- 喜欢开源、可定制、可自部署
- 有 API Key，不想额外付费

### 选 Copilot 如果你：
- 只需要代码补全和代码相关的聊天
- 重度使用 VS Code / GitHub 生态

### 选 Monica/Sider 如果你：
- 只需要简单的页面问答和翻译
- 不想自己配置 API Key
- 愿意为便利性付费
