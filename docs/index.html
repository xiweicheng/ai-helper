核心能力

不止是问答机器人，更是浏览器自动化引擎

🧠

ReAct 推理循环

LLM 思考 → 调用工具 → 结果反馈 → 继续推理。支持多轮工具调用、任务拆解、流式响应、三级反思质量保障。

🤖

多智能体协作

创建和管理多个自定义 AI 助手，内置代码审查、数据分析等模板。支持 Agent 子任务分发和并行协作。

🖱️

浏览器自动化

点击、填表、滚动、拖拽、截图、等待元素、下拉选择。用自然语言操控网页，无需写一行脚本。

📄

内容智能提取

纯文本、HTML、表格、链接、表单、图片、元数据。支持 Markdown/JSON 格式化输出，滚动收集长内容。

🖥️

代理本地操作

读写文件、搜索代码、执行终端命令。npm install、git 操作、运行脚本，直接在浏览器对话中完成。

🎯

Skill 技能系统

Workflow 和 Agent 两种技能类型，支持从对话沉淀技能。内置 skill-creator 元技能，一键创建可复用工作流。

🔌

MCP 协议扩展

支持 Model Context Protocol，连接第三方 MCP Server。自动发现工具、多 Server 并行，无限扩展能力边界。

📊

Markdown & 图表

代码高亮、Mermaid 流程图/时序图、表格导出 Excel。支持图表缩放、下载 PNG、复制源码。

📎

多模态文件问答

上传 PDF/Word/Excel/文本文件直接提问。浏览器端智能提取内容，连接 Agent 后支持上传至工作目录深度操作。

✏️

图片标注编辑器

内建画笔、矩形、椭圆、箭头、橡皮擦 6 种标注工具。支持颜色/粗细调节和 Ctrl+Z 撤销，编辑后直接发送。

👁️

视觉模型识图

对接视觉语言模型，支持截图识图对话。独立 API 配置，自动图片压缩减少 Token 消耗。

💡

上下文智能压缩

长引用内容自动摘要压缩，按 Token 数智能截断。三级压力监测，避免无关信息永久占据上下文空间。

💰

Token 统计面板

实时追踪 Token 消耗，会话/今日/历史多维度统计。趋势图展示、预算管理、上下文压力三级监测。

💾

会话导出/导入

批量导出对话为 .aihelper.json，支持完整消息历史、执行日志、质量评分。兼容新版/旧版格式导入恢复。

🔒

隐私与安全

API Key 本地存储。代理命令分级管控（黑名单拦截 + 灰名单确认），文件操作沙箱保护，配对认证。

30 秒快速开始

三步完成安装配置，即刻开始使用

1

加载扩展

下载项目后，在 chrome://extensions 开启开发者模式，加载 dist 目录。

git clone && npm install && npm run build
2

配置 API Key

右键扩展图标 → 选项，填入 DeepSeek API Key（或其他 OpenAI 兼容 API）。

API Key → 填入 → 保存
3

安装代理（可选）

如需本地文件读写和命令执行，全局安装代理服务并启动。

npm i -g ai-helper-agent && ai-helper-agent start
