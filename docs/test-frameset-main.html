Frameset 主内容面板

这是 Frameset 布局中的主内容区域，包含多种测试场景。

📝 测试指南：
1. 在本页面不同区域选中文字，测试工具栏显示
2. 在 Shadow DOM 区域选中文字
3. 在嵌套 iframe 中选中文字
4. 尝试双击和拖拽选中两种方式
📄 普通文本区域

这是普通的 HTML 文本内容，可以直接选中测试。Frameset 布局中每个 frame 都有独立的 document 对象，content script 会通过 all_frames 参数注入到每个 frame 中。

选中这段文字，测试划词工具栏是否正常出现，并能将内容带入输入框。
