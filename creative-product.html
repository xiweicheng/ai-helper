核心亮点

不止于工具调用

从输入到输出，全链路打磨每一个环节

01

多模态输入 — 图片 + 文件随心问

支持图片识别（Vision API）、PDF/Word/Excel 文件提取问答。内建图片标注编辑器，6 种标注工具，支持 Undo。有 Agent 时优先上传至工作目录供 AI 直接操作原始文件。

02

ReAct 推理循环 — 思考与行动的闭环

LLM 思考 → 决定调用工具 → 并行执行 → 结果反馈 → 继续推理。支持工具预筛选、Token 预算管理、上下文智能压缩、流式响应、多级超时控制和会话隔离取消。

03

Skill 系统 + MCP 协议 — 能力无限扩展

Workflow Skill 实现确定性自动化流程，Agent Skill 提供 AI 自主调用的知识扩展。MCP 协议支持动态注册第三方工具，多 Server 同时连接，工具自动合并。

04

划词浮动工具栏 — 无处不在的 AI 入口

选中任意网页文本即弹出毛玻璃风格工具栏，支持 AI 搜索、解释、翻译、总结、自定义提示词工具，可拖拽缩放的结果面板，让 AI 能力无缝融入浏览体验。

05

长期记忆 — 让 AI 越用越懂你

自动存储用户偏好、知识、决策记录和对话摘要。AI 在对话中主动调用记忆工具进行存储和检索，支持标签分类、重要性评分、智能压缩淘汰，让每次对话都更贴合你的需求。
